    • CSS Theme aims to style elements purely based on their semantic meaning or on the relationships between elements. However, it also provides a limited set of classes that can be used to create more advanced layouts.

    DomainClassEffect
    Containerscontainer-mediumSets the maximum size of the container to 800px.
    container-narrowSets the maximum size of the container to 1200px.
    container-wideSets the maximum size of the container to 1600px.
    LayoutrowTransforms its child elements into horizontally aligned columns.
    colInstructs an element to occupy as much space as possible. If all elements have this class they will all have equal width.
    col-NHorizontal space is divided in 12 equal columns. From col-1 to col-12 we can progressively specify columns of greater and greater width.
    Mobilehide-on-mobileHides an element if on small displays.
    hide-on-desktopHides an element if on large displays.
    disable-mobileDisable layout changes on small displays. It can be applied to elements that have the row class applied, nav bars, menus, etc to force them not to change their display on small screens.
    ColorsprimaryDepending on context, it changes background, text or border colors to match various hues derived from the theme's primary color.
    secondaryDepening on context, it changes background, text or border colors to match various hues derived from the theme's secondary color.
    successDepending on context, it changes background, text or border colors to match various hues derived from the theme's success color.
    errorDepending on context, it changes background, text or border colors to match various hues derived from the theme's error color.
    invertedTakes any primary, secondary, success or error color scheme and inverts it such that the background color is a lot more proeminent and the text color is usually a contrasting one.
    Alignmentalign-centerAligns elements centrally on the horizontal axis.
