CSS Theme aims to be plug & play as much as possible. In this section we've also seen a number of CSS classes that can be used to allow it to support many more cases.

Below is the full list of supported classes and the effect they produce:

ClassEffect
align-centerAligns elements centrally on the horizontal axis. See here.
cardTransforms an element into a card. See here.
circleRounds the edges off an element so it's a perfect circle. Useful for images. See here.
container-mediumSets the maximum size of the container to var(--width-container-medium). See here.
container-narrowSets the maximum size of the container to var(--width-container-narrow). See here.
container-wideSets the maximum size of the container to var(--width-container-wide). See here.
errorApplies an error-type style of an element by changing its background, foreground or text color. See here or here.
gapUsed in between col elements. Provides a gap between them so they can be separated into left and right sides. Has no effect on mobile displays. See here.
groupGroups various elements, like buttons or inputs, together, horizontally. See here.
hide-on-mobileHides an element if on small displays. See here.
hide-on-desktopHides an element if on large displays. See here.
rowTransforms child elements into horizontally aligned columns. Also the col-N class. See here.
successApplies a success-type style to an element by changing its background, foreground or text color. See here or here.
