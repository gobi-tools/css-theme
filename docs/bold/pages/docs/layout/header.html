    • A header is used to define the introductory content of a page or a section. The simplest top level header can contain a navigation element (nav or menu):

    <header>
  <nav>
    <ul>
      <li>
        <a href="...">
          Home
        </a>
      </li>
      <li>
        <a href="...">
          About
        </a>
      </li>
    </ul>
  </nav>
</header>
<main>
  <h1>Title</h1>
  <p>Lorem ipsum...</p>
</main>

    You create more complex "hero" layouts by placing an article inside a header.

    <!-- nav header -->
<header>
  <nav>
    <ul>
      <li>
        <a href="...">
          Home
        </a>
      </li>
    </ul>
  </nav>
</header>

<!-- hero header -->
<header>
  <article class="align-center">
    <h2>
      My blog
    </h2>
    <p>
      Lorem ipsum...
    </p>
    <form action="...">
      <div class="group">
        <input 
          type="email" 
          placeholder="..."/>
        <input 
          type="submit" 
          value="Subscribe"/>
      </div>
    </form>
  </article>
</header>
<main>
  <h1>Title</h1>
  <p>Lorem ipsum...</p>
</main>

    headers and asides combine together to form a "banner" element that can be placed at the top of a page or mid-content.

    <main>
  <header>
    <aside>
      <div class="group">
        <div class="row">
          <div>
            ...
          </div>
          <div class="gap">
          </div>
          <button>
            ...
          </button>
        </div>
      </div>
    </aside>
  </header>
</main>
