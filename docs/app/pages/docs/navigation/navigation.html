The most basic navigation element is created by placing an unordered list within a nav element. It's suitable as the top level navigation for a document, where each item can be a link to a different page.

  • Item 1
  • Item 2
  • Item 3

Links may contain icons to enhance the look and feel of the navigation bar as well as collapsible items.

  • Home
  • Docs
  • More
    • Option 1
    • Option 2

Navigtion items may use the aria-selected attribute to denote they are selected.

  • Selected
  • Unselected
<header>
  <nav>
    <ul>
      <!-- simple nav item -->
      <li>
        <a href="...">Item 1</a>
      </li>
      <!-- selected item -->
      <li aria-selected>
        <a href="...">Item 2</a>
      </li>
      <!-- nav item with icon -->
      <li>
        <a href="">
          <svg ...></svg>
          <span>Item 3</span>
        </a>
        <!-- collapsible navigation tree -->
        <ul>
          <li>
            <a href="...">Option 1</a>
          </li>
          ...
        </ul>
      </li>
    </ul>
  </nav>
</header>

You can use other elements such as select, input, img, etc to add more functionality to the navigation bar.

  • Home
  • Docs

<li>
  <img height="24" src="..." alt="logo" />
</li>
<li>
  <select>...</select>
</li>
<li>
  <input type="search" placeholder="Search..."/>
</li>

Navigation can be split into a left and a right section by placing a gap list item.

  • Home
  • Menu

<!-- left side -->
<li>...</li>
<li>...</li>

<!-- gap -->
<li class="gap"></li>

<!-- right side -->
<li>...</li>
<li>...</li>

Navigation can be combined with the card class to create a more striking display.

  • Home
  • Docs

<nav class="card">
  <ul>
    <li>...</li>
    <li>...</li>
    <li>...</li>
    <li class="gap">...</li>
    <li>...</li>
  </ul>
</nav>

Finally, the navigation bar is responsive. On large displays it expands horizontally and on smaller displays it switches to a vertical layout.

<header>
  <nav>
    <ul>
      <li>
        <a href="...">
          <svg ...></svg>
          <span>Home</span>
        </a>
      </li>
      <li>
        <a href="...">
          <svg ...></svg>
          <span>Docs</span>
        </a>
      </li>
      <li>
        <a href="...">
          <svg ...></svg>
          <span>More</span>
        </a>
        <ul>
          <li><a href="...">Option 1</a><li>
          <li><a href="...">Option 2</a><li>
        </ul>
      </li>
    </ul>
  </nav>
</header>
